# M5Stack Core 页面切片 v0.2

本文档记录 M5Stack Core Basic 上的第一版可用交互切片。目标是让 320x240、无触摸、三按键的 M5 Core 能完成基础导航、WiFi 配置和常用设置。

## 设计原则

- 主屏保持一眼可读：当前呼号、RX 状态、站点、电池、WiFi 优先。
- 三按键优先承担焦点移动和确认：A/C 移动焦点，B 确认。
- 文本输入暂时复用 LVGL 软键盘。焦点在键盘上时，A/C 改为左右移动键位，B 输入当前键。
- 设置页保留原有配置能力，所有可点击控件都应进入 LVGL 焦点组。
- 需要退出时，移动焦点到“返回”或关闭按钮后按 B。

## 全局按键约定

| 操作 | 普通控件焦点 | 键盘焦点 |
| --- | --- | --- |
| A | 上一个控件 | 左移键位 |
| B | 确认 / 点击 | 输入当前键 |
| C | 下一个控件 | 右移键位 |

## 页面 1：主屏 Home

用途：日常显示 FMO 状态。

```text
┌────────────────────────────────────────┐
│  WiFi  电量  21:32              设置态 │
├────────────────────────────────────────┤
│                                        │
│              BI8SIG                    │
│             当前通联                   │
│                                        │
├────────────────────────────────────────┤
│ 当前站点  成都中继 / FMO-01            │
│ 上次通联  BG5ESN  QSO 128             │
├────────────────────────────────────────┤
│ A 上一个      B 确认        C 下一个   │
└────────────────────────────────────────┘
```

按键：

- A/C：移动主屏可点击控件焦点。
- B：确认当前焦点控件。移动到设置按钮后按 B 进入设置页。

## 页面 2：设置菜单 Settings

用途：替代触摸操作，集中完成 WiFi、FMO、呼号、音量、背光、电池和站点配置。

```text
┌────────────────────────────────────────┐
│ 设置                             v1.x  │
├────────────────────────────────────────┤
│ > FMO 服务器                           │
│   呼号                                 │
│   WiFi 设置                            │
│   音量 / 背光                          │
│   电池 / 站点                          │
│   返回主屏                             │
├────────────────────────────────────────┤
│ A 上移        B 确认        C 下移     │
└────────────────────────────────────────┘
```

按键：

- A/C：移动焦点。
- B：进入或执行。
- 返回：移动焦点到返回按钮后按 B。

## 页面 3：收藏站点 Station

用途：三按键切换收藏站点。当前优先复用现有站点弹窗逻辑。

```text
┌────────────────────────────────────────┐
│ 收藏站点                    1/3  在线  │
├────────────────────────────────────────┤
│ > 成都 FMO-01                          │
│   北京 FMO-02                          │
│   上海 FMO-03                          │
│   广州 FMO-04                          │
│   深圳 FMO-05                          │
├────────────────────────────────────────┤
│ A 上一个      B 切换        C 下一个   │
└────────────────────────────────────────┘
```

按键：

- A/C：移动站点焦点。
- B：切换到选中站点。
- 上一页 / 下一页：移动焦点到对应按钮后按 B。

## 页面 4：文本输入 Keyboard

用途：在无触摸屏的 M5 Core 上复用 LVGL 软键盘。

```text
┌────────────────────────────────────────┐
│ WiFi 密码                              │
├────────────────────────────────────────┤
│ SSID: HomeWiFi                         │
│ PASS: ********                         │
├────────────────────────────────────────┤
│ q w e r t y u i o p                    │
│ a s d f g h j k l                      │
│ 1# z x c v b n m 退格 确认             │
├────────────────────────────────────────┤
│ A 左移        B 输入        C 右移     │
└────────────────────────────────────────┘
```

处理策略：

- 输入框获得焦点时显示软键盘。
- 焦点移动到软键盘后，A/C 在键位之间移动，B 输入当前键。
- WiFi 扫描选中 AP 后自动回到 WiFi 设置页，并把焦点移到密码输入框。

## 第一阶段实现范围

第一阶段不重做所有设置细节，只做能让 M5 Core 用起来的核心闭环：

1. M5 Core ILI9341 兼容显示驱动和引脚配置。
2. 禁用 XPT2046 触摸，启用 A/B/C 按键输入。
3. LVGL 焦点组覆盖主屏、弹窗和设置页可点击控件。
4. 软键盘焦点下 A/C 左右选键，B 输入。
5. WiFi 扫描选中 AP 后焦点自动回到密码框。

## 需要后续验证

- 不同批次 M5 Core 的 LCD 方向和颜色是否一致。
- A/B/C 按键是否需要针对个别硬件版本调整消抖。
- GPIO25 内置 DAC 音量是否足够，是否需要按 M5 扬声器电路补充功放控制。
- 老版本 M5 Core 如果只有 4MB Flash，当前分区和固件大小是否还能容纳。
